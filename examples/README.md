This directory contains a curated collection of machine learning notebooks covering both supervised and unsupervised ML techniques. The notebooks are designed to emphasize conceptual understanding, algorithm behavior, and practical application through hands-on experimentation with real and synthetic datasets.

---

## Supervised Learning

The supervised learning notebooks focus on learning predictive models from labeled data. These notebooks cover both regression and classification tasks and illustrate how different model families make assumptions about data structure, complexity, and generalization.

**Topics include:**

- Linear regression
- Logistic regression
- k-Nearest Neighbors (kNN)
- Perceptron
- Regression trees
- Ensemble methods
- Neural networks

---

## Unsupervised Learning

The unsupervised learning notebooks focus on discovering structure in unlabeled data. These methods are used for clustering, dimensionality reduction, and understanding latent patterns without explicit targets.

**Topics include:**

- K-Means clustering
- DBSCAN
- Principal Component Analysis (PCA)
- Community detection

---

## Datasets

The notebooks use a mix of publicly available real-world datasets (e.g., housing data, financial data, graph data) and synthetic datasets generated for visualization and clarity. Each notebook documents the dataset it uses and any preprocessing steps applied.

---

## Requirements

Most notebooks rely on standard Python scientific computing libraries:

- Python 3.9+
- NumPy
- Pandas
- scikit-learn
- Matplotlib / Seaborn
- NetworkX (for graph-based methods)
